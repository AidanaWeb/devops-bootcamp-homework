7. **Подсчет количества слов:**
   Напишите скрипт, который будет принимать текстовый файл в качестве аргумента и подсчитывать количество слов в этом файле.

```
#!/bin/bash

file=$1

words=$(wc -w < "$file")
echo "$words"
```
