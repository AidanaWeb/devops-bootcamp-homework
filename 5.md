5. **Обработка текстового файла:**
   Создайте текстовый файл с несколькими строками текста. Напишите скрипт, который будет читать файл и выдавать каждую строку в обратном порядке.

```
#!/bin/bash

file=$1

while IFS= read -r line; do
	echo $line | rev
done < $file
```
